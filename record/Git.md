# Git

## Git 基础命令

- git add .
- git commit -m "message"
- git push origin master
- git log
- git branch xxx
- git checkout xxx
- git merge


## Git 高级命令

- git rebase master 合并分支, merge 也可以
- git stash / git stash pop 暂存现场/恢复现场
- git reflog 查看 HEAD 移动记录
- git reset 重置 Git 记录
